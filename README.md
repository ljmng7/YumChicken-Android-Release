# 馋香鸡 安卓版

把厨房变简单，从今天开始做一顿。

馋香鸡 安卓版 是一款专注于“做饭过程”的极简菜谱工具。它不是追求海量内容的菜谱大全，而是把每日推荐、分类检索、清晰步骤、收藏整理和随烹推荐放在一个干净的界面里，帮助你更轻松地走进厨房。

## 功能亮点

- 每日推荐：每天给出适合下厨的菜，减少“今天吃什么”的纠结。
- 清晰菜谱：按分类浏览和搜索菜品，结构简单，快速开始烹饪。
- 详细步骤：配料和步骤统一排布，跟做时更容易看清。
- 收藏整理：把常做、爱吃的菜留在身边，想做时马上找到。
- 随烹推荐：根据冰箱里的食材组合，看看可以做出什么好味道。

## 设计理念

馋香鸡关注的不是“菜有多少”，而是“做饭是否更轻松”。应用保持轻量、清爽、少打扰的体验，减少无关信息，让用户把注意力留给食材、步骤和灶台前的当下。
